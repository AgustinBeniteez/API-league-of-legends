Documentación / Introducción

Introducción

Esta es una API robusta y fácil de usar construida con Node.js y Express que proporciona información actualizada sobre los campeones de League of Legends (LoL) y Teamfight Tactics (TFT).

Tiempo Real

Sincronización automática con la última versión de parches de Data Dragon (Riot Games).

Formato Amigable

Datos transformados en arrays limpios y fáciles de iterar en cualquier frontend.

¿Por qué usar esta API?

  • Privacidad: No recolectamos datos de usuario.
  • Rapidez: Servidor optimizado con baja latencia.
  • Código Abierto: Cualquiera puede auditar o contribuir al proyecto.

Guía de Uso

Implementar la API en tu proyecto es sencillo. Puedes usar fetch nativo de JavaScript o cualquier cliente HTTP.

Cómo realizar una petición

Para obtener los datos de un campeón específico, como Lux, simplemente realiza una petición GET al endpoint correspondiente:

fetch('https://api-league-of-legends-ruby.vercel.app/lol/champion/Lux')
  .then(response => response.json())
  .then(data => {
    console.log("Nombre:", data.name);
    console.log("Título:", data.title);
    console.log("Imagen:", data.image.full);
  });

Recorriendo la respuesta JSON

Al obtener la lista completa de campeones, recibirás un Array que puedes iterar fácilmente para mostrar tarjetas o listas en tu interfaz:

// Ejemplo recorriendo todos los campeones
const champions = await fetch('https://api-league-of-legends-ruby.vercel.app/lol/champions').then(r => r.json());

champions.forEach(champ => {
  console.log(`Campeón: ${champ.name} - ${champ.title}`);
});

Endpoints de LoL

Endpoints principales para acceder a la base de datos de League of Legends. La URL base es https://api-league-of-legends-ruby.vercel.app.

GET https://api-league-of-legends-ruby.vercel.app/lol/champions

Obtiene todos los campeones con estadísticas e imágenes.

GET https://api-league-of-legends-ruby.vercel.app/lol/champion/:id

Detalle de un campeón por nombre o ID (ej: /lol/champion/Lux).

GET https://api-league-of-legends-ruby.vercel.app/lol/items

Lista completa de todos los objetos del juego.

Hechizos de Invocador

Endpoints para obtener información sobre los hechizos de invocador (Summoner Spells).

GET https://api-league-of-legends-ruby.vercel.app/lol/spells

Obtiene todos los hechizos disponibles en el juego.

GET https://api-league-of-legends-ruby.vercel.app/lol/spell/:id

Detalle de un hechizo por ID o nombre (ej: /lol/spell/SummonerFlash).

Endpoints de TFT

Datos específicos para Teamfight Tactics (Set actual).

GET https://api-league-of-legends-ruby.vercel.app/tft/champions

Lista completa de campeones, costes y tiers.

GET https://api-league-of-legends-ruby.vercel.app/tft/champion/:id

Detalle específico de una pieza de TFT.

Contribución

Si deseas contribuir al desarrollo o ejecutar el proyecto localmente, sigue estos pasos.

1. Clonar el repositorio

git clone https://github.com/AgustinBeniteez/API-league-of-legends.git
cd API-league-of-legends

2. Instalar dependencias

npm install

3. Ejecutar el servidor

node index.js

El servidor se iniciará por defecto en http://localhost:3000 para pruebas locales.
